# Contributing to Hello Open Source

Thank you for your interest in contributing to the Hello Open Source project! This guide will help you understand how to contribute effectively.

## Getting Started

1. **Fork the repository** - Click the "Fork" button at the top right of the GitHub page.
2. **Clone your fork** - `git clone https://github.com/YOUR-USERNAME/hello-open-source.git`
3. **Create a branch** - `git checkout -b your-branch-name`

## Contribution Options

### 1. Add Your Profile

Create a new file in the `contributors` directory named after your GitHub username (e.g., `contributors/username.md`). Use the template in `templates/contributor-template.md` as a guide.

### 2. Add a Recipe

Share your favorite recipe by adding a new file to the `recipes` directory. Use the template in `templates/recipe-template.md` as a guide.

### 3. Hello World Program

Add a "Hello World" program in your favorite programming language to the `hello-world` directory. Name your file according to the language used (e.g., `java-hello-world.java`).

### 4. Fix or Improve

Found a typo or think something could be explained better? Make the change and submit a pull request!

## Submitting Your Contribution

1. **Commit your changes** - `git commit -m "Add [brief description]"`
2. **Push to your fork** - `git push origin your-branch-name`
3. **Create a pull request** - Go to your fork on GitHub and click "New Pull Request"

## Code of Conduct

- Be respectful and inclusive
- Help others learn
- Focus on constructive feedback
- Remember this is a learning environment

## Review Process

All contributions will be reviewed by project maintainers. We may suggest changes or improvements before merging.

Thank you for contributing to open source!
